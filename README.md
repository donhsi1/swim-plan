# 🏊 自由泳提速训练方案

> 一套系统化的自由泳进阶训练计划，帮助游泳爱好者突破速度瓶颈，提升技术水平。

## 在线预览

- **训练方案（网页版）**: https://donhsi1.github.io/swim-plan/

## 项目内容

| 文件 | 说明 |
|------|------|
| `index.html` | 自由泳提速训练方案（完整幻灯片网页版，离线可用） |
| `自由泳提速训练方案.pdf` | 训练方案 PDF 版，可下载打印 |
| `dog-comic.html` | 彩蛋：旺财 & 小白的搞笑四格漫画 🐶 |
| `ppt-web/` | 幻灯片 Web 应用源文件 |

## 训练方案概览

本训练方案涵盖以下核心模块：

- **技术基础**：手型、入水角度、划水路径优化
- **体能训练**：有氧耐力、无氧爆发力专项提升
- **节奏控制**：划频与划幅的最佳平衡点
- **转身技术**：滚翻转身标准动作分解
- **呼吸节奏**：双侧呼吸训练，提升水感与稳定性
- **阶段计划**：8 周系统训练进阶路线图

## 使用方式

### 网页版（推荐）

直接访问在线链接，或下载 `index.html` 后双击在浏览器中打开：

- 按键盘 `←` `→` 方向键翻页
- 点击左右箭头按钮翻页
- 支持完全离线使用，无需网络

### PDF 版

下载 `自由泳提速训练方案.pdf` 即可在本地查看或打印。

## 技术栈
